# Dotfiles

A collection of configuration files for a streamlined Linux and Termux environment.

## 📂 Repository Structure

- `bashrc/`: Shell configurations (`.bashrc`).
- `vim/`: Editor settings (`vimrc`).
- `colorschemes/`: Themes for the terminal.
- `styles/`: Themes for the Mousepad text editor.
- `fontconfig/`: Custom font configurations and rules.
- `fonts/`: Preferred Terminus font.
